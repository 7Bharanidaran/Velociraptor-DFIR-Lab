# Velociraptor DFIR Lab

## Overview

This repository documents my hands-on experience with Velociraptor for Digital Forensics and Incident Response (DFIR) and Threat Hunting on Kali Linux.

## Environment

- Operating System: Kali Linux
- Tool: Velociraptor v0.76.6

## Objectives

- Install and configure Velociraptor
- Set up the Velociraptor server
- Create an administrator account
- Generate and deploy a Linux client package
- Enroll the client successfully
- Build a DFIR and Threat Hunting lab
